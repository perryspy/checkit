
## Getting started
1. `npm install`
2. `mongod`
  * starts the local database
3. `npm start`

## Getting started - Dev
1. `npm install`
  * installs all needed npm modules, bower packages, and does an initial build of our JS and CSS files.
2. `mongod`
  * starts the local database
3. `gulp`
  * starts the server using nodemon
  * this in turn watches files as well
